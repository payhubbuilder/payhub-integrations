=== Description ===
PayHub's plugin for aMember membership software.  This plugin allows you to process credit card payments through aMember with PayHub.  Please note, a PayHub account is required through Central Payment to process transactions.

=== How to find your API credentials ===
1. Log into PayHub's Virtual Terminal
2. Click on Admin
3. Under General heading, click on 3rd Party API.
4. Copy down your Username, Password, and Terminal Id.  Please note the username and password is case sensitive.

=== Installation ===
(this assumes you have aMember installed)
1. Download the zip file.
2. Place the zip file in the '/application/cc/plugins/' directory.
3. Unzip the contents of the zip file in the directory.
4. Once the plugin is installed, log into aMember.
5. Click on Setup/Configuration, then click on the Plugins tab.
6. Under Payment Plugins, select the PayHub plugin, and save.
7. Now click on the PayHub tab.
8. Fill in the required fields with your API credentials.
9. Check mark Test Mode if you want to send transactions to our test server.  Leave it uncheck marked if you want to do transactions to our production servers.
